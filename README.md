## MCP Servers

AgentNxt Chat includes 28+ MCP (Model Context Protocol) servers. See [MCP Servers](docs/mcp-servers.md) for the full list.

## Getting Started

1. Sign in at [auth.agnxxt.com](https://auth.agnxxt.com) with your AgentNxt Auth credentials
2. Access any service — SSO keeps you logged in across all `*.agnxxt.com` services
3. Start with [AgentNxt Chat](https://chat.agnxxt.com) to chat with AI using all available tools
