# Camunda

Decision automation and business process management engine.

**URL:** https://camunda.agnxxt.com

## Features

- BPMN 2.0 process modeling and execution
- DMN decision tables
- Form builder for human tasks
- REST API for process management
- Operate, Tasklist, and Modeler UIs

## Configuration

- Docker compose: `/home/ubuntu/platform/docker-compose.yaml` (service: `camunda`)
- Port: 8080 (internal)
