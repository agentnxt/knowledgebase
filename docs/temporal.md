# Temporal

Workflow orchestration engine for durable, long-running processes.

**URL:** https://temporal.agnxxt.com (UI)

## Features

- Durable workflow execution with automatic retries
- Workflow versioning and migration
- Activity heartbeats and timeouts
- Cron schedules for recurring workflows
- Web UI for workflow management

## Configuration

- Docker compose: `/home/ubuntu/platform/docker-compose.yaml` (services: `temporal`, `temporal-ui`)
- UI port: 8080 (internal)
