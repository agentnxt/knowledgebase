# AgentNxt Search

Privacy-respecting meta search engine. Built on SearXNG.

**URL:** https://search.agnxxt.com

## Features

- Aggregates results from 70+ search engines
- No tracking, no ads, no user profiling
- JSON API for agent/programmatic use
- Image, news, files, and map search

## Configuration

- Settings: `/home/ubuntu/platform/searxng/settings.yml`
- Instance name: `AgentNxt Search`

## API Usage

```bash
curl "https://search.agnxxt.com/search?q=your+query&format=json"
```
